#Visual Odometry Benchmark

this repository contains a research oriented implementation of visual odometry methods for computer vision and robotics.

## Motivation
Visual odometry is a core problem in autonomous systems, robotics and self driving technology. the goal of this project is to implement, compare and evaluate classical and learning-based VO methods.

## Scope
-Classical feature-based VO (ORB, FAST)
-Evaluation on public datasets

## Tech Stack
- Python
- OpenCV
- NumpPy
- Matplotlib
